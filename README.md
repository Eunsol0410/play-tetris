# Play Tetris

## 기능 요구사항
- 키보드 입력에 따라 동작을 구분한다.
  - LEFT, RIGHT, DOWN : 해당 방향으로 떨어지는 조각이 움직인다.
  - SPACE : 조각이 아래로 움직일 수 없을 때까지 수직으로 떨어진다.
  - UP : 조각이 시계방향으로 90도 회전한다.
- 조각의 모양과 색상은 랜덤하게 나타난다.
- 난이도에 따라 조각이 떨어지는 속도를 달리한다.
- 블럭이 한 줄 가득 채우면 줄을 제거하고 점수를 올린다. 
- 제거된 줄의 개수에 따라 가산점을 부여한다.
- 라인 10줄이 제거되면 난이도가 1 증가한다.
- 쌓여있는 조각이 최상단에 닿게 되면 게임이 종료된다.
- 다음에 나올 조각을 미리 볼 수 있도록 한다.
- 게임 세팅은 시작 레벨, 배경음악 재생여부, 다크모드를 선택할 수 있다.
- 다크모드는 시스템 세팅을 기준으로 기본 값을 설정한다.
- 게임이 진행 중에 레벨을 설정하면 이전 게임은 종료된다.
